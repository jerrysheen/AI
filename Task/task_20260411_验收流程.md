## 二、验收标准

### 2.1 通用验收标准
每个平台都必须满足：

1. **接口存活**: 能正常导入和调用主函数
2. **任务创建**: 能通过 info-grab-manager 添加任务
3. **状态流转**: `raw` → `pending` → `processed`
4. **索引生成**:
   - daily_jobs.json 正确更新
   - job_timeline.json 记录完整事件
   - 平台 index.json 正确生成
5. **数据落盘**:
   - metadata.json 存在且有实际内容
   - content.txt 存在且有实际内容
   - 视频文件存在 (如有视频)
   - 图片目录存在 (如有图片)
   - transcript.txt 存在 (如有视频且转写成功)

### 2.2 平台特定验收标准

#### B站 (pull-bilibiliInfo)
- 测试用例: 无字幕视频，验证ASR转写
- 验收: 视频下载成功 + ASR转写成功（或触发回落机制）

#### YouTube (pull-youtubeInfo)
- 测试用例: 有字幕视频，验证字幕获取
- 验收: Metadata获取 + 字幕获取成功 (视频下载可选)

#### Twitter (pull-Twitter)
- 测试用例1: 纯图文推文
- 测试用例2: 含视频推文
- 验收: 图文下载 + 视频下载 + ASR转写

#### 小红书 (pull-xhs)

- 验收: 图文下载 + 视频下载 + ASR转写
